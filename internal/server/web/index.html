API 文档
发送文本消息
向微信发送文本通知，中文内容需用文件方式传递避免编码损坏。
echo '{"text":"通知内容"}' > /tmp/msg.json
curl -s -X POST http://localhost:0/api/v1/wechat-bot/send/text \
  -H "Content-Type: application/json" \
  --data-binary @/tmp/msg.json
返回: {"status":"sent"} 或 {"status":"buffered"}
发送图片 / 文件 / 视频
curl -s -X POST http://localhost:0/api/v1/wechat-bot/send/image \
  -H "Content-Type: application/json" \
  -d '{"file_path":"C:/photo.png"}'
替换 /send/image 为 /send/file 或 /send/video
查询收到的消息
curl -s http://localhost:0/api/v1/wechat-bot/messages
支持增量拉取: ?since=2026-07-16T10:00:00Z，清空用 DELETE
查询状态
curl -s http://localhost:0/api/v1/wechat/status
端口自动检测，无需手动替换，查看 ~/.wechat-bot-server/config.json
